Bienvenido a mi Portafolio

Aquí presento mis proyectos y aprendizajes en el desarrollo web. En esta sección encontrarás información sobre mis habilidades, proyectos realizados y el progreso en mi formación como desarrollador web.

Mi objetivo es crear páginas web con estructura clara, estilo visual atractivo y experiencia de usuario óptima.

Proyectos Destacados

Fundamentos de HTML

Proyecto inicial que demuestra el dominio de etiquetas semánticas, estructura de documentos HTML, y buenas prácticas de organización de contenido. Incluye navegación entre páginas, formularios y uso correcto de rutas relativas.

  • Etiquetas semánticas
  • Formularios interactivos
  • Navegación estructurada
  • Multimedia integrada
Nivel: Principiante

Introducción a CSS

Aplicación de estilos CSS para mejorar la presentación visual. Trabajo con selectores, propiedades de color, espaciado, bordes y tipografía. Uso de clases e identificadores para reutilización de estilos y mantenimiento eficiente del código.

  • Selectores CSS
  • Propiedades de estilo
  • Flexbox y Grid
  • Diseño responsivo
Nivel: Intermedio

Diseño Web Semántico

Implementación de HTML5 semántico con el uso correcto de etiquetas como article, section, aside, figure y footer. Esta aproximación mejora la accesibilidad, el SEO y la mantenibilidad del código para proyectos profesionales.

  • HTML5 semántico
  • Accesibilidad web
  • Optimización SEO
  • Mejores prácticas
Nivel: Avanzado

Detalles de Proyectos

¿Cuál fue el primer proyecto?

Mi primer proyecto fue crear una página web personal completa con múltiples secciones, navegación entre páginas, formularios y contenido multimedia. Este proyecto sentó las bases para entender la estructura HTML5.

¿Cuáles son mis objetivos futuros?

Deseo aprender JavaScript para interactividad, dominar frameworks como React, entender backend con Node.js, y eventualmente crear aplicaciones web completas. También quiero especializarme en diseño responsivo y accesibilidad web.

¿Qué herramientas utilizo?

Utilizo Visual Studio Code como editor principal, Git para control de versiones, y navegadores modernos para testing. Complemento con Font Awesome para íconos y librerías CSS externas cuando es necesario.
